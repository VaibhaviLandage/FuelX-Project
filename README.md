# 🚗 Fuel Delivery System (Community Project)

## 📌 Description
This project is a web-based Fuel Delivery System that allows users to order fuel online and get it delivered to their location. It provides a simple and efficient way to manage fuel requests, payments, and delivery tracking.

---

## 🎯 Features
- User Registration & Login
- Fuel Order Request System
- Select nearest petrol pump
- Real-time Order Tracking
- Secure Payment Page
- Admin Dashboard for managing orders
- User Profile Management
- Customer Support Page

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (based on package.json)
- Database: MySQL

---

## 📂 Project Structure
- `index.html` – Home page  
- `login.html` – User login  
- `Register.html` – User registration  
- `dashboard.html` – User dashboard  
- `request.html` – Fuel request page  
- `payment.html` – Payment page  
- `tracking.html` – Order tracking  
- `admin.html` – Admin panel
